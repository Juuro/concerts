### HIGH (1 root cause — 3 instances)

**Missing timeouts on all GitHub API fetch calls**

All external HTTP calls to GitHub lack `AbortSignal.timeout()`. A stalled GitHub API response will hang the route handler or cron job until Vercel's function timeout kills it.

| File | Line | Function |
|------|------|----------|
| `src/lib/github/feedback-issues.ts` | 51 | `createFeedbackIssue` |
| `src/lib/github/fetch-github-issue-state.ts` | 12 | `fetchGithubIssueStateByNumber` |
| `src/lib/github/project-v2.ts` | 19 | `githubGraphql` |

**Fix:** Add `signal: AbortSignal.timeout(10_000)` to each `fetch()` call.
**Contributors:** Resilience, Security
